# CODEXSUN Ecommerce

Commerce application for CODEXSUN.

This repository owns its backend modules, frontend modules, migrations, workers, permissions, navigation contributions, and tests. It integrates with the generic Platform through the public CodexsunApp contract.

The repository is currently an installable boundary; those runtime roles are added only when
real Ecommerce modules exist. Read `assist/AGENT-GUIDE.md` before changing it.
